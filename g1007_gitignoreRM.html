Utilizando o gitignore e git rm

Para evitar bagunça com arquivos de configuração e módulos que não precisam ser versinados.

Gitignore global

Vc pode criar um arquivo para ignorar globalmente determinados arquivos independente do repositório que vc está trabalhando.

Para verificar se vc tem o gitignore global, verifique a pasta digitando no console.

Se houver, editar o arquivo, caso contrário pode criar e editar usando o vim

Dentro do arquivo vc pode escrever o que ddeseja ignorar globalmente, vc pode consultar o repositorio do proprio GitHub que dá uma idéia do que ignorar em https://github.com/github/gitignore
